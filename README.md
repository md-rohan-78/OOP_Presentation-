# ☁️ Java & Cloud Computing — Group Presentation

An interactive web-based presentation on **Java and Cloud Computing**, built with HTML, CSS, and JavaScript.

## 🔗 Live Site

[View Presentation](https://md-rohan-78.github.io/)

---

## 📁 Project Structure

```
├── index.html     # Main HTML structure (17 slides)
├── style.css      # All styling and animations
└── script.js      # Nav dots and scroll progress logic
```

## 📋 Topics Covered

1. Overview of Cloud Computing & Java
2. Presentation Objectives
3. Basics of Cloud Computing (IaaS, PaaS, SaaS)
4. Importance of Java in Cloud
5. Java Cloud Frameworks (Spring Cloud, Micronaut, Quarkus)
6. Microservices Architecture
7. Spring Cloud Components (Eureka, Hystrix, Ribbon)
8. Java & Microservices Integration
9. Deployment on AWS
10. Deployment on Google Cloud (GCP)
11. Cloud Security Practices
12. Netflix Case Study
13. Key Lessons & Future Trends
